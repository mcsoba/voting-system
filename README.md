# Szavazási Rendszer

Ez a projekt egy egyszerű szavazási rendszert valósít meg, amely lehetővé teszi szavazások létrehozását, szavazatok rögzítését és eredmények megtekintését.

## Funkciók

- Új szavazások létrehozása dátummal, témával, típussal és elnök nevével.
- Szavazatok rögzítése különböző képviselők részéről az igen, nem vagy tartózkodó lehetőségek közül.
- Szavazások eredményeinek lekérdezése az elnök, a témakör és a típus alapján.
- Napra szűrés: Adott napon megtartott és regisztrált szavazások eredményeinek lekérdezése.
- Képviselők számának, szavazatok számának és eredményeknek a megtekintése egy adott szavazásnál.

## Használt Technológia
- Java Spring Boot
- Spring Data JPA
- H2 adatbázis
- Thymeleaf sablonmotor
